Single-Cell RNA Sequencing Analysis

At the Migal Research Institute, I was the first student to undertake a computational single-cell RNA sequencing analysis project.
I developed a comprehensive pipeline (R) for future researchers, encompassing all stages of data analysis.

This project involved analyzing large datasets on gene expression in three different rats to examine the effect of EPA lactone in the context of hypertension. My work included data quality control, dimensionality reduction, cell identification via clustering, various visualizations, and identifying differences between treatments.
Feel free to visit my GitHub and delve into my project.

Macro explanation

Classification of cells into clusters using UMAP, each point represents a gene expression profile of a cell. Color sorting was based on gene expression similarity.

Pf4 is a genetic marker for immune system cells (encoded for a cytokine in blood platelets).

Based on genetic markers I discovered the cell types in each cluster.

Volcano plot shows that lactone treatment reduces the effect of hypertension on gene expression in endothelial cells.
